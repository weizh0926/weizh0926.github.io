Field teams live in **drawings, submittals, and change orders**. A useful agent does not hallucinate specs: it cites sheet numbers, links to the RFI log, and flags when a schedule conflict is already resolved in the last bulletin. The hard part is **trust**—superintendents need provenance, not fluent guesses. Start with retrieval over controlled document sets, then widen scope only when monitoring shows stable precision.
